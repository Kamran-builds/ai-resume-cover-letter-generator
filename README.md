# 📄 AI Resume & Cover Letter Generator

An AI-powered app that generates professional resumes and 
cover letters using LLaMA 3.3 70B via Groq API.

## 🚀 Live Demo
[Click here to try the app](https://huggingface.co/spaces/kamran26/Ai-resume-generator)

## 🛠️ Tech Stack
- LLM: Groq API (LLaMA 3.3 70B)
- UI: Gradio
- Deployment: Hugging Face Spaces
- Prompt Tool: Claude (Anthropic)

## ✨ Features
- Generates ATS-friendly resumes
- Creates personalized cover letters
- Uses prompt chaining (resume → cover letter)

## 📦 How to Run
pip install groq gradio
# Add your GROQ_API_KEY
python app.py
